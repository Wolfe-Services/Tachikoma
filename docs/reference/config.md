## Environment Variables

| Variable | Description | Default |
|----------|-------------|---------|
| `NODE_ENV` | Environment mode | `development` |
| `RUST_LOG` | Rust logging level | `info` |
| `TACHIKOMA_CONFIG_PATH` | Config file path | `./config.toml` |

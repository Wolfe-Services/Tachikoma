# Getting Started with Tachikoma

This guide will help you get up and running with Tachikoma quickly.

## Prerequisites

- Node.js 18+ 
- Rust 1.75+
- Git

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-org/tachikoma
   cd tachikoma
   ```

2. Install dependencies:
   ```bash
   npm run install:all
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

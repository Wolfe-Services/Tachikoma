## FAQ Entry Template
```markdown
### Q: How do I reset my configuration?

**A:** To reset your configuration to defaults:

```bash
tachikoma config reset
```

This removes your user configuration file at `~/.config/tachikoma/config.yaml` and restores default settings.

**Note:** This does not affect project-level configurations.

**See also:**
- [Configuration Reference](./config-reference.md)
- [Troubleshooting](./troubleshooting.md)
```
